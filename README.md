## Krita
A free painting software used for concept art, 2D animation, and photoshop.
Perks

* FREE! 
* Built in Layering system and brush presets
* Cross platform
  
Drawbacks

* Large file crashes
* Tool limitations, Not as powerful or accurate as Photoshop
* Small amounts of Latency
